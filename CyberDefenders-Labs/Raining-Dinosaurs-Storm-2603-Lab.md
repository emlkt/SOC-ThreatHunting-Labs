# Raining Dinosaurs - Storm-2603 Lab

### Категория: Threat Hunting

### Инструменты: Detect It Easy (DiE), pyinstxtractor-ng, pylingual, CyberChef, Splunk, логи Suricata.

## Сценарий

17 ноября 2025 года в сети зафиксировали подозрительную активность: сначала аномальный трафик с сервера в DMZ, потом движение внутри сети и в конце — массовое шифрование файлов. Моя задача была восстановить полную цепочку атаки: от первого взлома до финального ущерба. В распоряжении были подозрительный exe-файл и логи в Splunk.

## Initial Access

### Q1. Эта уязвимость позволила злоумышленнику получить первоначальный доступ и позже переместиться на DC01. Какой CVE ID связан с этим эксплойтом?

**Ответ:** `CVE-2025-59287`

1. Открыла подозрительный файл `ping.exe` в Detect It Easy. DiE сразу показал, что это не системная утилита, а упакованный PyInstaller python скрипт. 
2. Вытащила содержимое через `pyinstxtractor-ng.exe` и получилась папка с .pyc файлами. Среди них был файл `cve-2025-59287.pyc`. 
3. Декомпилировала его через `pylingual`, посмотрела код и нашла подтверждение: внутри была логика эксплуатации именно этой уязвимости WSUS.

## Exfiltration

#### Q15. Проанализируйте сетевые журналы (suricata) на предмет подключений с DC01. Сколько файлов было успешно загружено на сервер злоумышленника?

**Ответ:** `262`

1. Искала логи Suricata. 
2. Искала исходящие соединения с DC01 на внешний ip злоумышленника 18.193.76.209 + фильтр по статусу HTTP 201 (Created) - файл загружен успешно. 
3. Посчитала уникальные URI в запросах (каждый файл имел свой путь).

`index=* sourcetype="suricata:*" src_ip="10.10.11.*" dest_ip="18.193.76.209" http.status=201 | stats dc(http.uri) as Unique_Files`

<img width="550" height="110" alt="15вопрос" src="https://github.com/user-attachments/assets/1b276801-0705-4cbb-9379-70a30bf9c2f1" />

#### Q16. Какой идентификатор техники MITRE ATT&CK лучше всего описывает метод эксфильтрации, использованный злоумышленником?

**Ответ:** `T1048.003`

(Злоумышленник передавал файлы через обычный HTTP (метод PUT) на свой сервер с портом 4000. При этом основной C2-канал был через Velociraptor. Согласно матрице MITRE, использование отдельного веб-протокола для передачи данных, который отличается от основного канала C2 - это **T1048: Exfiltration Over Alternative Protocol** с **.003** **Exfiltration Over Unencrypted/Obfuscated Non-C2 Protocol** (в данном случае HTTP). Это позволяет злоумышленникам обходить мониторинг c2 каналов, маскируя кражу данных под обычный веб-трафик.)
